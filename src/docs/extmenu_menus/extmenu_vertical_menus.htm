Menu description

Check out the Code.zikula project page for more information and the download section.
There are vertical and horizontal 1 level menus. The menus have a hover and selected state. The Home menu item is pre-selected to show the effect. The menus use a single-image rollover for the hover and selected state. For example the image of menu vertical22:

is shifted to the correct position using the CSS backgroung position property. It is, in this case, 200px wide and 3 x 32px = 96px high. The image is loaded in one call.
When the user has admin rights extra links are shown for adding the current URL to the menu and editing the menublock. You can see this in the right vertical menu.

Scalability of the menus:

  • Vertical menus 1, 3-6, 21 are fully scalable in width
  • Vertical menus 2, 7-9, 11, 15, 18-20, 23 are only scalable upto 300px. The maximum width is determined by the imagewidth. If you need only a smaller size menu, you could reduce the image width to reduce bandwidth.
  • Vertical menus 10, 12-14, 16, 17, 22 are not scalable and have a fixed width of 200px. If you want to adjust this you will have to edit the style sheet as well as adapt the images that belong to the menu.
Choosing the menus is done by choosing an alternative style sheet. This is based upon: A List Apart - Alternative Style
