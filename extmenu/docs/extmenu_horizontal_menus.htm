Menu description

Check out the Code.zikula project page for more information and the download section.
There are vertical and horizontal 1 level menus. The menus have a hover and selected state. The Home menu item is pre-selected to show the effect. The menus use two single-image rollover for the hover and selected state (not every image also has a selected). The sliding doors effect (see A List Apart article) is used for the dynamic width of the horizontal menu items. The maximum width of the right side image is 175px. So if you have large menuitems, you will have to enlarge the right side images. For example the images of menu horizontal17:

right side image:
They are shifted to the correct position using the CSS backgroung position property.
When the user has admin rights extra links are shown for adding the current URL to the menu and editing the menublock. You can see this in the lower horizontal menu.
Choosing the menus in this demo file is done by choosing an alternative style sheet. This is based upon: A List Apart - Alternative Style
